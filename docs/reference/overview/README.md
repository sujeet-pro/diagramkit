# Overview

diagramkit exposes four entry points — the main API (`diagramkit`), utilities (`diagramkit/utils`), color processing (`diagramkit/color`), and SVG-to-raster conversion (`diagramkit/convert`). The [CLI reference](/reference/cli) covers commands and flags, [Configuration](/reference/config) covers all config fields, [API](/reference/api) covers the programmatic interface, and [Types](/reference/types) lists all exported TypeScript types.
